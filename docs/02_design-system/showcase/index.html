О системе

Единый источник правды для визуала VELO: слоистые токены (Primitives → Semantic → Component), бренд-мотив (мандала + знак Θ), шрифт Marmelad, каталог компонентов и паттернов. Цвета/размеры берём только из токенов (--velo-*), без хардкода. Шесть разделов — см. навигацию выше.
