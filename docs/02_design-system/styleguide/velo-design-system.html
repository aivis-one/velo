Universal app background — absolutely-positioned velo-bg-app.png. Every user screen uses this except SuccessScreen (pure white).

Centered mini-mandala header (login / register / oauth). 154px mandala, 37px top padding.

.top-header

Detail view header — glass back-arrow (absolute left) + centered title. Used on all nested screens.
